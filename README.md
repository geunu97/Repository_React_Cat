<h1 align="center">고양이 사진관</h1>
<p align="center"><a href="https://github.com/geunu97/Repository_VanillaJS_Cat"> VanillaJS 고양이 사진관</a>을 React 고양이 사진관으로 전환하여 개발하였습니다.</p>

## 사용 스택

- 언어: Javascript <br>
- 라이브러리: React <br>
- 스타일링: Emotion <br>
- 자동화: Github Actions <br>
- 배포: Netlify <br>

## 작업 내용
 
- 내부 코드 전환, 추가 작업 및 자동화 작업
